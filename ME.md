## Portfolio Website Features & Requirements

I want a premium multi-page portfolio website designed to attract clients and convert visitors into real opportunities.

### 1. Overall Design & Aesthetic

- Tech-focused blue color theme with clean white balance  
- Professional, modern interface  
- Fully responsive design optimized for mobile users in Nigeria  
- Use only Lucide React icons  
- No emojis  
- Use real and relevant images showing coding environments, dashboards, and AI tools  
- Include a personal image in the About section for trust and credibility
